## Notes

- HTML mockups are static and can be viewed directly in the browser
- React mockups (`.tsx`) require the app to be running
- The `dist/public/mockups/` folder contains built versions (don't edit these)
- When creating new mockups, place HTML files in `client/public/` or `client/public/mockups/`
